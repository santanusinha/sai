## Common Use Cases

SAI can be used for:

### Code Review

```bash
java -jar target/sai-1.0-SNAPSHOT.jar --input "Review the changes in my last commit"
```

Use SAI to get AI-powered code review feedback.

### Repository Analysis

```bash
java -jar target/sai-1.0-SNAPSHOT.jar --input "Summarize this repository structure"
```

Get an overview of any codebase.

### Documentation Generation

```bash
java -jar target/sai-1.0-SNAPSHOT.jar --input "Generate API documentation for this project"
```

Create documentation from your code.

### Interactive Development

```bash
java -jar target/sai-1.0-SNAPSHOT.jar
```

Use SAI as a coding assistant during development.

### CI/CD Integration

```bash
# In your pipeline
echo "Analyze test failures" | java -jar sai.jar > analysis.txt
```

Integrate SAI into your automated workflows.
